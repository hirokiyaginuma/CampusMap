# Interactive Campus Map

## Required Libraries
* Folium - pip instal folium
* PyQt5 - pip install PyQt5
* PyQtWebEngine - pip install PyQtWebEngine
* Selenium - pip install Selenium
* BeautifulSoup - pip install bs4
